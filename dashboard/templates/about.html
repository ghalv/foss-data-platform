Core Components

Apache Iceberg

Table Format for the Lakehouse

Provides ACID transactions, schema evolution, time travel, and partitioning for data lakes. Enables reliable, high-performance analytics on object storage.

ACID Transactions Time Travel Schema Evolution
Trino

Distributed SQL Query Engine

Fast, distributed SQL engine for querying data lakes and databases. Enables unified analytics across multiple data sources with standard SQL.

Distributed SQL Multi-Source High Performance
dbt

Data Transformation Tool

Transforms raw data into analytics-ready tables using SQL. Enables data engineers to build reliable, tested data pipelines.

SQL Transforms Testing Documentation
MinIO

S3-Compatible Object Storage

High-performance object storage with S3 API compatibility. Provides scalable, cost-effective storage for data lakes.

S3 Compatible High Performance Scalable
Hive Metastore

Metadata Catalog Service

Centralized metadata repository for Iceberg tables. Provides table discovery, schema management, and access control.

Metadata Catalog Schema Management Discovery
PostgreSQL

Metadata Database

Reliable database backend for Hive Metastore. Ensures metadata persistence and transactional consistency.
